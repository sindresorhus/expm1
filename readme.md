# expm1

> ES2015 [`Math.expm1()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/expm1) [ponyfill](https://ponyfill.com)

## Install

```
$ npm install expm1
```

## Usage

```js
const expm1 = require('expm1');

expm1(1);
//=> 1.718281828459045
```
